Hexoはじめました

  • hexo
2015-09-25

初記事という事で、手慣らしのためHexo + Github Pagesの導入方法を書いていきます。

ちょうど一年前位、学習のアウトプットにHexoでブログを書き始めようと考えドキュメントを漁っていましたが、なぜか画像が表示されなかったりなどの理由で放置状態。

屁糞hanakus…:(´ཀ`」 ∠):ｱﾊﾞﾊﾞﾊﾞ

JekyllやOctpressなどのツールも試したりしましたが、どうしてもHexoのyillaというテーマが使いたかったため、試行錯誤、ようやく導入に漕ぎ着けました。

導入（インストール〜デプロイ）までの手順は以下です。
既に完了している場合は、適宜飛ばして下さい。

1. Githubでリポジトリを作成しておく

以下の名称でリポジトリを作成してください。
「アカウント名.github.io.git」

例として、当ブログのリポジトリを掲載します↓
https://github.com/taka-p/taka-p.github.io

頑張って作りましょう。（雑ですみません）

2. Node.jsのインストール

まずは、Hexoを動作させるNode.jsを本家サイトからダウンロード→インストールしましょう。
GUIでインストールできるので楽チンですね。
http://nodejs.jp/nodejs.org_ja/
